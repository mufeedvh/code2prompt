`Code2prompt` cuenta con una interfaz de usuario de terminal (TUI) completamente interactiva. Esta guía le guía a través del lanzamiento de la TUI y el dominio de sus cinco pestañas principales.
</Card>

## Lanzar la TUI

Para abrir la interfaz interactiva, simplemente agregue la bandera `--tui` a su comando junto con la ruta de su proyecto:

```bash
code2prompt ruta/a/su/código-base --tui

```

---

## 1. Pestaña de Selección (Árbol de archivos)

La pestaña de **Selección** es donde organiza el contexto exacto que desea enviar al LLM. Muestra una vista jerárquica de su proyecto, respetando sus reglas `.gitignore` y `.c2pconfig`.

* **Navegación:** Use `↑` / `↓` para mover el cursor, y `←` / `→` para expandir o contraer directorios.
* **Selección:** Presione `Espacio` para seleccionar o deseleccionar un archivo o un directorio completo.
* **Modo de búsqueda:** Presione `s` o `/` para ingresar al modo de búsqueda.
* Puede escribir texto o usar comodines (`*`, `?`) para filtrar instantáneamente el árbol.
* Presione `Enter` o `Esc` para salir del modo de búsqueda y volver a la navegación normal.

## 5. Pestaña de Salida

Una vez que haya seleccionado sus archivos, ajustado su configuración y elegido su plantilla, presione `Enter` para generar la salida final.

La pestaña de **Salida** muestra el prompt final exactamente como se enviará al LLM.

* **Desplazar:** Use `↑` / `↓`, `PgUp` o `PgDn` para revisar el texto generado.
* **Copiar:** Presione `c` para copiar instantáneamente todo el prompt a su portapapeles del sistema.
* **Guardar:** Presione `s` para guardar el prompt directamente en un archivo Markdown local (por ejemplo, `prompt_20260613_204753.md`).

Si se da cuenta de que olvidó un archivo, simplemente presione `1` para volver a la pestaña de Selección, marque el archivo y presione `Enter` para regenerar la salida en tiempo real.
