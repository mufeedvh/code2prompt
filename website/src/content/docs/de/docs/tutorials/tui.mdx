## TUI starten

Um die interaktive Oberfläche zu öffnen, fügen Sie einfach die Flagge `--tui` zu Ihrem Befehl zusammen mit dem Pfad zu Ihrem Projekt hinzu:

```sh
code2prompt pfad/zu/ihrem/codebase --tui
```

---

## 1. Registerkarte **Auswahl** (Dateibaum)

Die Registerkarte **Auswahl** ist dort, wo Sie den genauen Kontext zusammenstellen, den Sie an das LLM senden möchten. Sie zeigt eine hierarchische Ansicht Ihres Projekts und respektiert dabei Ihre `.gitignore`‑ und `.c2pconfig`‑Regeln.

* **Navigation:** `↑` / `↓` zum Bewegen des Cursors, `←` / `→` zum Erweitern bzw. Reduzieren von Verzeichnissen.  
* **Auswahl:** Leertaste, um eine Datei oder ein ganzes Verzeichnis auszuwählen bzw. abzuwählen.  
* **Suche:** `s` oder `/`, um in den Suchmodus zu wechseln. Sie können Text eingeben oder Platzhalter (`*`, `?`) nutzen, um den Baum sofort zu filtern.  
* `Enter` oder `Esc`, um den Suchmodus zu verlassen und zur normalen Navigation zurückzukehren.

## 4. Registerkarte **Vorlage**

Die Registerkarte **Vorlage** ist dort, wo die Magie passiert. Sie präsentiert ein mehrspaltiges, anpassbares Layout, das Ihnen hilft, die perfekte Handlebars‑Vorlage zu erstellen.

Wenn Sie sich in der Registerkarte **Vorlage** befinden, können Sie mit Tastenkombinationen sofort zwischen drei Unter‑Panels wechseln:

## 5. Registerkarte **Ausgabe**

Nachdem Sie Ihre Dateien ausgewählt, die Einstellungen angepasst und die Vorlage gewählt haben, drücken Sie `Enter`, um die endgültige Ausgabe zu erzeugen.

Die Registerkarte **Ausgabe** zeigt das Prompt exakt so, wie es an das LLM gesendet wird.

* **Scrollen:** `↑` / `↓`, `PgUp` oder `PgDn`, um den generierten Text zu prüfen.  
* **Kopieren:** `c`, um das gesamte Prompt sofort in Ihre System‑Zwischenablage zu kopieren.  
* **Speichern:** `s`, um das Prompt direkt in einer lokalen Markdown‑Datei zu speichern (z. B. `prompt_20260613_204753.md`).

Falls Ihnen eine Datei fehlt, drücken Sie einfach `1`, um zurück zur Registerkarte **Auswahl** zu springen, markieren Sie die Datei und drücken Sie `Enter`, um die Ausgabe in Echtzeit neu zu generieren.
