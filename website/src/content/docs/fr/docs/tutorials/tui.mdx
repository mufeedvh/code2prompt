## Lancer le TUI

Pour ouvrir l'interface interactive, ajoutez simplement le flag `--tui` à votre commande avec le chemin de votre projet :

```bash
code2prompt chemin/vers/votre/base-de-code --tui

```

---

## 1. Onglet Sélection (Arborescence de fichiers)

L'onglet **Sélection** est l'endroit où vous organisez le contexte exact que vous souhaitez envoyer au LLM. Il affiche une vue hiérarchique de votre projet, respectant vos règles `.gitignore` et `.c2pconfig`.

* **Navigation :** Utilisez `↑` / `↓` pour déplacer le curseur, et `←` / `→` pour développer ou réduire les répertoires.
* **Sélection :** Appuyez sur `Espace` pour sélectionner ou désélectionner un fichier ou un répertoire entier.
* **Mode recherche :** Appuyez sur `s` ou `/` pour entrer en mode recherche.
* Vous pouvez taper du texte ou utiliser des caractères génériques (`*`, `?`) pour filtrer instantanément l'arborescence.
* Appuyez sur `Entrée` ou `Échap` pour quitter le mode recherche et revenir à la navigation normale.

## 5. Onglet Sortie

Une fois que vous avez sélectionné vos fichiers, ajusté vos paramètres et choisi votre template, appuyez sur `Entrée` pour générer la sortie finale.

L'onglet **Sortie** affiche le prompt final exactement tel qu'il sera envoyé au LLM.

* **Défiler :** Utilisez `↑` / `↓`, `PgUp` ou `PgDn` pour parcourir le texte généré.
* **Copier :** Appuyez sur `c` pour copier instantanément l'intégralité du prompt dans votre presse-papiers système.
* **Sauvegarder :** Appuyez sur `s` pour enregistrer le prompt directement dans un fichier Markdown local (par exemple, `prompt_20260613_204753.md`).

Si vous réalisez que vous avez manqué un fichier, appuyez simplement sur `1` pour revenir à l'onglet Sélection, cochez le fichier et appuyez sur `Entrée` pour régénérer la sortie en temps réel.
