## Запуск TUI

Для открытия интерактивного интерфейса добавьте флаг `--tui` к команде вместе с путём к проекту:

```sh
code2prompt путь/к/вашему/codebase --tui
```

## 1. Вкладка **Выбор** (дерево файлов)

- **Навигация:** `↑` / `↓` перемещают курсор, `←` / `→` раскрывают/сворачивают каталоги.  
- **Выбор:** `Space` выбирает/снимает выбор файла или целой директории.  
- **Поиск:** `s` или `/` переводит в режим поиска; вводите текст или шаблоны (`*`, `?`) для мгновенной фильтрации.  
- **Выход:** `Enter` или `Esc` закрывает режим поиска и возвращает к обычной навигации.

## 3. Вкладка **Статистика**

Перед отправкой большого промпта в LLM проверьте ваш токен‑бюджет.

`←` / `→` переключает между тремя представлениями.

## 4. Вкладка **Шаблон**

Здесь происходит магия. Вы получаете адаптивный многоколоночный макет для создания идеального Handlebars‑шаблона.

В режиме **Шаблон** используйте клавиатурные комбинации для быстрого переключения между тремя подпанелями.

<img src="/assets/images/tui4.png" alt="Вкладка Шаблон" style="width: 100%;" />

## 5. Вкладка **Вывод**

После выбора файлов, настройки и шаблона нажмите `Enter` для генерации окончательного вывода.

- **Прокрутка:** `↑` / `↓`, `PgUp`, `PgDn` для просмотра сгенерированного текста.  
- **Копировать:** `c` копирует весь промпт в системный буфер обмена.  
- **Сохранить:** `s` сохраняет промпт в локальный Markdown‑файл (например, `prompt_20260613_204753.md`).

Если вы забыли добавить файл, нажмите `1`, вернитесь к вкладке **Выбор**, отметьте файл и нажмите `Enter` для мгновенного пересчёта.
