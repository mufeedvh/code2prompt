## Запуск TUI

Чтобы открыть интерактивный интерфейс, просто добавьте флаг `--tui` к вашей команде вместе с путём к проекту:

```sh
code2prompt путь/к/вашему/codebase --tui
```

## 1. Вкладка **Выбор** (дерево файлов)

Эта вкладка позволяет собрать точный контекст, который будет отправлен LLM. Она отображает иерархическое представление вашего проекта, учитывая правила `.gitignore` и `.c2pconfig`.

- **Навигация:** `↑` / `↓` перемещают курсор, `←` / `→` раскрывают/сворачивают каталоги.  
- **Выбор:** `Space` выбирает/снимает выбор файла или целой директории.  
- **Поиск:** `s` или `/` переводит в режим поиска; вводите текст или шаблоны (`*`, `?`) для мгновенной фильтрации.  
- **Выход:** `Enter` или `Esc` закрывает режим поиска и возвращает к обычной навигации.

Здесь происходит магия. Вы получаете многострочный адаптивный макет, помогающий создавать идеальный Handlebars‑шаблон.

При нахождении в этой вкладке используйте клавиатурные комбинации для мгновенного переключения между тремя подпанелями:

## 5. Вкладка **Вывод**

После выбора файлов, настройки и шаблона нажмите `Enter`, чтобы сгенерировать окончательный вывод.

- **Прокрутка:** `↑` / `↓`, `PgUp` или `PgDn` для просмотра сгенерированного текста.  
- **Копировать:** `c` копирует весь промпт в системный буфер обмена.  
- **Сохранить:** `s` сохраняет промпт в локальный Markdown‑файл (например, `prompt_20260613_204753.md`).

Если вы забыли добавить файл, нажмите `1`, вернитесь к вкладке **Выбор**, отметьте файл и нажмите `Enter` для мгновенного пересчёта.
